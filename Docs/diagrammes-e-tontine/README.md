# Diagrammes E-Tontine actualises

Ce dossier contient les diagrammes UML actualises du projet E-Tontine.

Le fichier principal est :

- `../diagrammes-uml-e-tontine-actualises.md`

Les sources individuelles `.mmd` et les rendus `.svg` peuvent etre generes a partir de ce document. Les diagrammes ont ete limites aux vues necessaires pour comprendre clairement le fonctionnement global de l'application dans un memoire de fin d'etude :

- cas d'utilisation global ;
- cas d'utilisation administrateur ;
- cas d'utilisation membre ;
- activite d'authentification ;
- activite de gestion d'un cycle ;
- sequences des flux importants ;
- diagramme de classes simplifie ;
- MCD simplifie ;
- architecture applicative ;
- deploiement.

Chaque diagramme est disponible en trois formats :

- `.mmd` : source Mermaid editable ;
- `.svg` : rendu vectoriel ;
- `.png` : image utilisable directement dans le memoire Word.
