# Diagrammes Draw.io - E-Tontine

Ce dossier contient une version `.drawio` pour chaque diagramme academique PNG.

Chaque fichier s ouvre dans diagrams.net / Draw.io et contient le diagramme PNG correspondant comme objet image. Vous pouvez donc deplacer, redimensionner, annoter ou recomposer le diagramme dans Draw.io. Pour modifier chaque trait ou chaque libelle separement, il faudra le redessiner dans Draw.io a partir de cette base visuelle.

Les PNG prets a inserer dans le memoire sont dans `../diagrammes-academiques-e-tontine/`.
